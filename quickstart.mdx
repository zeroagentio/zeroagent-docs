Welcome to ZeroAgent! This Quickstart guide will help you seamlessly connect to GitHub and start automating your tasks. 
Follow these simple steps to get started:

### Step 1: Access the ZeroAgent Console
- **Log in** to your ZeroAgent account. 

### Step 2: Select GitHub
- Check the **Tools** available in sidebar, select `GitHub` from the list of integrations.

### Step 3: Click the Setup Button
- Click the **Setup** button shown in the page. This action will prompt you to initiate the authorization process.

### Step 4: Authorize ZeroAgent

- You will be redirected to the GitHub authorization page.
- **Log in** to your GitHub account.
- Review the permissions requested by ZeroAgent and click **Authorize** to grant access.

### Step 5: Verify Connection

- After authorization, you will be redirected back to ZeroAgent.
- A confirmation message will appear, indicating that GitHub has been successfully connected.
- You can also check the status of the connection on the top. 

### Step 6: Start Using Your AI Agent

- Now that GitHub is connected, you can start using your AI agent to automate tasks.
- For example, you can type the following prompt to create a new GitHub repository:
- Prompt: "Create a GitHub repo named 'ZeroAgent-Demo' with a description 'This is my new repository'"
- Your AI agent will handle the request and create the repository for you.
