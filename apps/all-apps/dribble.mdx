## Supported Actions

The following table outlines the supported actions and example prompts for our Typeform integration:

1. ### Project Management

| Action                 | Description                                                                   | Example Prompt                              |
| :--------------------- | :---------------------------------------------------------------------------- | :------------------------------------------ |
| **Create a new project** | Creates a new project with a given title | `Create a new 'my design portfolio' project` |
| **List all projects** | Displays a list of all projects the user has access to | `Show me all my projects` |

2. ### Shot Management

| Action                            | Description                                                               | Example Prompt                                 |
| :-------------------------------- | :------------------------------------------------------------------------ | :--------------------------------------------- |
| **Show my shots** | Displays a list of all shots the user has uploaded | `Show me all my shots` |
| **Find a shot** | Searches for a specific shot by shot Id | "Find the 'Sasquatch' shot" |


3. ### Job Posting

| Action                            | Description                                                               | Example Prompt                                 |
| :-------------------------------- | :------------------------------------------------------------------------ | :--------------------------------------------- |
| **Post a job**	| Creates a new job posting with a given title, description, and requirements	| `Post a job for a 'UI/UX designer'` |
