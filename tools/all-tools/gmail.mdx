## Supported Actions
The following table outlines the supported actions and example prompts for our Typeform integration:

1. ### User Management

| Action          | Description                                         | Example Prompt                                                 |
| :-------------- | :--------------------------------------------------- | :---------------------------------------------------------------- |
|  **Get User Details**  | Retrieves the authenticated Gmail user's details        | `Get my Gmail account information` |


2. ### Email Management

| Action          | Description                                         | Example Prompt                                                 |
| :-------------- | :--------------------------------------------------- | :---------------------------------------------------------------- |
|  **Create a Draft Email**  | Creates a new draft email        | `Draft an email to John Doe regarding 'Sick leave'` |
|  **Send an Email**  | Sends a composed email       | `Send email to Jane Doe with regarding 'Sick leave` |
|  **List Email Drafts**  | Fetches a list of all draft emails.      | `List all my draft emails` |
